EE 128




The scope of this project is to implement a working lock utilizing the K64F microcontroller to turn a stepper motor by receiving a pulse width modulation signal from the Arduino microcontroller using input capture.


The Arduino provides a pulse width modulation signal based on the number of correct code inputs.


The inputs are determined using a potentiometer to set a number which is displayed on a 7 segment display and a button to enter the number.


Once the correct combination is entered, the corresponding pulse width modulation signal unlocks the lock on the K64F.


The K64F then counts down from ten until it automatically locks and resets the Arduino.


The countdown is displayed on another 7 segment display.
